Product Designers: Yoonjo Choi
Duration: 2020~Current

Overview

Current DSP platform has significant UI debt due to several mergers, rapid growth and feature teams without design support. An audit to capture the many UI/UX mishaps and inconsistencies was a necessity in order to plan for a smooth transition to the new visual styles and components.

Audit, IA & Priority List

Audit: Total 360 pages

Information Architecture

Priority List

Priority to address ranked by severity.
Severeness is measured on 5 dimensions; Usage Data, Satisfaction Report (internal score), Visual, Usability and Scale.

Audit Example — Data Table

Current Issue: Row Highlight
  • Incorrect Row-Highlight states due to unnecessary row interaction
  • Selected-Checkbox gets lost because it is the same color as Row-Highlight
Current Issue: Wide Tables & Checkbox Inconsistency
  • When Table-Width exceeds Browser-Width it extends the page width, resulting in a horizontal scroll that breaks the UI
  • Low discoverability of Customize-Column Selector due to being pushed to the end of tables that exceed the browser width
  • Inconsistent Checkboxes under Customize-Column Selector

Component Update — Data Table

  • Remove Click-Row interaction
  • Change Row-Highlight color
  • Horizontal scroll to occur within the table so table overflow does not effect page width
  • Keep Customize-Column selector sticky to the right side of the browser so it is visible at all times
  • Update Checkbox styles under Customize-Column Selector to keep component consistent

Reflection

Data-tables make up more than 50% of the UI and amending this component elevates the platform in cohesiveness to a great degree. Additionally, addressing inconsistency further paves the way to easy adoption of the up-coming new style guide.

UI systems and inconsistency is an ongoing effort and will always be a topic to keep an eye out for; not just as an individual but for the team of UX designers and UI engineers alike. It is important to communicate the efforts and direction with the team and bring awareness to spotting and addressing inconsistencies.
