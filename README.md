# Yoonjo Choi — Portfolio

Product design portfolio site for [yoonjochoi.com](https://www.yoonjochoi.com).

Static HTML site built with Bootstrap 4 and custom CSS.

## Local preview

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Structure

- `index.html` — Homepage
- `project/` — Professional project case studies
- `cc/` — Creative corner projects
- `css/`, `js/`, `img/` — Assets
